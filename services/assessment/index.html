Security Assessment

Comprehensive evaluation of your infrastructure, applications, and security posture. Find your gaps before attackers do.

What we assess

  • Network infrastructure — Firewalls, routers, switches, exposed ports, DNS configuration
  • Endpoint security — Workstations, servers, mobile devices, IoT assets
  • Application security — Code review, dependency scanning, API endpoint analysis
  • Identity & access management — User accounts, privilege escalation paths, MFA coverage
  • Data protection — Encryption at rest and in transit, key management, data classification
  • Cloud configuration — S3 buckets, IAM policies, security groups, exposed services
  • Supply chain — Third-party vendors, software dependencies, SBOM analysis

Our approach

🔍 On-device first
We run assessment tools locally on your infrastructure. No data leaves your network during the assessment. Findings are delivered as encrypted reports.

Phase 1: Discovery

Full asset inventory. We map every device, service, and entry point in your network. No agent installation required — passive discovery first, active scanning with your approval.

Phase 2: Vulnerability Analysis

Every asset is tested against known CVEs, misconfigurations, and weak defaults. We use open-source tools (Nmap, OpenVAS, Lynis) plus our own custom checks developed for Vigilar's threat engine.

Phase 3: Risk Scoring

Vulnerabilities are scored using CVSS + business context. A CVE on your public-facing web server scores higher than the same CVE on an isolated internal machine. You get a prioritized action plan, not a list of problems.

Phase 4: Report & Roadmap

Encrypted report delivered via ARCZT Vault sharing (end-to-end encrypted). Includes executive summary, technical findings, risk scores, remediation steps, and a 90-day hardening roadmap.

Deliverables

  • Executive summary (PDF, encrypted)
  • Technical findings report with CVSS scores
  • Asset inventory spreadsheet
  • 90-day remediation roadmap
  • Vigilar deployment plan for your team (optional)

Engagement model

  • One-time assessment — Full audit, delivered in 2-3 weeks
  • Quarterly assessment — Recurring audits with delta reporting
  • Continuous assessment — Vigilar deployed to endpoints for real-time posture monitoring + quarterly manual review
